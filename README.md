# 🌳 Progress Grove 3.1

Персональный трекер обучения iOS-разработке.

## Изменения 3.1

- убрана отдельная колонка «Подтянуть»;
- после завершения теории и практики рядом с темой появляется чёрная галочка;
- нажатие на галочку переводит тему в состояние «Нужно повторить» и показывает красный восклицательный знак;
- нажатие на восклицательный знак возвращает галочку и впервые отмечает тему как закреплённую;
- каждая тема учитывается в статистике закрепления только один раз;
- статистика показывает:
  - темы, которые требуют повторения;
  - количество разных закреплённых тем;
- добавлены достижения за 1, 10, 25 и 50 закреплённых тем;
- в настройках появилась кнопка сброса активности за сегодня;
- сохранена миграция данных из предыдущих версий;
- мобильная версия остаётся адаптированной под карточки тем.
